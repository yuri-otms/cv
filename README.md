## 業務内容

### 2023年9月〜2025年9月：　企業・自治体等のコーポレートサイト開発

会社独自のCMSまたはWordPressを使用したコーポレートサイト

#### 担当業務

お知らせ、業務実績等のCMS開発、及びお問い合わせフォーム実装を担当

#### 発揮したバリュー

- 会社独自のCMSについて、不具合修正や機能追加・仕様変更を、営業と相談しながら改善提案・実装を行い、運用面も含めた品質向上に貢献。
- プルリクエストやコードレビューの文化がなかった環境において、CMS開発チームの立ち上げを提案し、チーム内でのレビュー運用を定着させ、開発品質と属人化防止に貢献。
- PL不在・PM多忙な環境下で自ら手を挙げて進捗管理を担当し、全案件を可視化するため案件一覧表を作成。
- 営業・PMと連携し、案件発生時に早期共有される連絡フローを構築・標準化。
- 仕様・期日・インフラ情報を事前整理し、開発前に仕様確認や改善提案を実施。
- 開発段階での手戻りを削減し、短期案件でも余裕を持った進行とリードタイム短縮に貢献。
